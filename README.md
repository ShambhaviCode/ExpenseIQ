# ExpenseIQ

A modern expense tracking dashboard built with HTML, CSS, JavaScript, and Chart.js. ExpenseIQ helps users track income, manage expenses, monitor financial health, and visualize spending patterns through an intuitive and responsive interface.

## Features

- Add income and expense transactions
- Delete transactions
- Real-time balance calculation
- Income and expense summaries
- Interactive Chart.js analytics
- Transaction search functionality
- Local Storage persistence
- Responsive design

## Tech Stack

- HTML5
- CSS3
- JavaScript (ES6)
- Chart.js
- Local Storage API

## Getting Started

Clone the repository:

```bash
git clone https://github.com/ShambhaviCode/ExpenseIQ.git
```

Open `index.html` in your browser.

## How It Works

1. Add income or expense transactions.
2. View real-time updates to balance, income, and expenses.
3. Analyze spending patterns using the dashboard chart.
4. Search transactions instantly.
5. Data is automatically saved in Local Storage.
